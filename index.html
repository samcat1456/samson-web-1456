// CircuitForge — smart sensor example
#include <DHT.h>
#include <WiFiS3.h>
 
DHT sensor(2, DHT22);
 
void setup() {
  Serial.begin(9600);
  sensor.begin();
  WiFi.begin(SSID, PASS);
}
 
void loop() {
  float temp = sensor.readTemperature();
  uploadToCloud(temp); // ✓ live
}
 
→ Uploading to cloud... SUCCESS
→ Temp: 21.4°C

Meet Samson

Hey, I'm Samson — founder of CircuitForge and the engineer behind every project we take on. I've been building with Arduino for 3–5 years, and what started as curiosity turned into a full-on passion for making hardware actually do things.

Some of my builds include an automatic motorbike indicator system and an ultrasonic distance tracker — both built from scratch with Arduino. I also do 3D printing, which means I can design and fabricate custom enclosures and parts for any project, not just the electronics.

When I'm not at the workbench, you'll find me at the gym. That same discipline — showing up, putting the work in, improving every session — is exactly how I approach every build.
